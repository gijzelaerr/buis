Buis
====

A webfrontend for toil


Requirements
============

 * python3
 * virtualenv
 * rabbitmq

on OSX with homebrew:
```
$ brew install python3 rabbitmq pyenv-virtualenv
$ brew services start rabbitmq
```

on debian/ubuntu:
```
$ sudo apt-get install rabbitmq-server python3 virtualenv
```
